무료 GitHub PAT 토큰 발급 프로세스

대시보드와 GitHub 무료 컴파일러를 연동하는 필수 단계

1단계: GitHub 계정 생성 및 로그인

무료 저장소 및 컴파일러 대여를 위해 우선 GitHub 공식 홈페이지에 가입하고 로그인해 주세요.

2단계: 개발자 설정 메뉴 진입

로그인 후 우측 상단의 본인 프로필 아이콘 클릭 ➔ [Settings] 클릭 ➔ 왼쪽 메뉴 최하단의 [Developer Settings]를 차례대로 눌러서 이동합니다.

3단계: Classic 토큰 생성

[Personal access tokens] 선택 ➔ [Tokens (classic)] 선택 ➔ 우측 상단 [Generate new token] 버튼 클릭 ➔ [Generate new token (classic)]을 선택합니다.

4단계: 권한 및 설정 세팅 (가장 중요)

  • Note (이름): 임의로 NPMake_Build_Key 등을 기입합니다.
  • Expiration (만료일): 무제한으로 사용하고 싶다면 [No expiration]으로 지정합니다.
  • Select scopes (권한): 아래 2개 항목은 반드시 체크해야 빌드 명령이 전달됩니다:

5단계: 발급 및 대시보드 등록

페이지 최하단의 [Generate token] 버튼을 클릭합니다. 생성된 ghp_로 시작하는 긴 비밀문구를 복사하여 대시보드 [GitHub & 컴파일 설정] 탭의 토큰 입력칸에 붙여넣기 하면 연결이 완료됩니다!
